JusBrasil Busca Inteligente

Use Inteligencia Artificial para encontrar jurisprudencia relevante no JusBrasil com precisao e eficiencia.

1 Configurar API Key

Voce precisa de uma API key da Anthropic para usar a IA.
Acesse console.anthropic.com/settings/keys e crie uma key.

A key sera configurada diretamente no painel quando voce clicar no bookmarklet no JusBrasil (passo 2). Nao precisa de servidor local!

2 Instalar o Bookmarklet

Arraste o botao abaixo para a barra de favoritos do seu navegador:

JusBrasil IA
Arraste este botao para a barra de favoritos

Se a barra de favoritos nao estiver visivel: Cmd+Shift+B (Mac) ou Ctrl+Shift+B (Windows)

3 Como usar

  1. a Abra o JusBrasil Jurisprudencia e faca login com sua conta
  2. b Clique no bookmarklet "JusBrasil IA" na barra de favoritos
  3. c Um painel vai aparecer no lado direito da tela
  4. d Descreva seu caso em linguagem natural e clique "Pesquisar"
  5. e A IA gera as queries, busca automaticamente e pontua os resultados
